# BilimQuest AI LMS

Интерактивный прототип LMS-платформы для начальной школы по теме:

> Цифрлық және AI технологияларын интеграциялау арқылы функционалдық және логикалық сауаттылықты дамыту.

## Что внутри

- отдельная стартовая страница ученика с игровыми предметными мирами;
- отдельные страницы предметов, урока, теста и AI-разбора результата;
- отдельный вход и кабинет учителя;
- уроки по предметам: математика, қазақ тілі, әдебиеттік оқу, жаратылыстану, дүниетану;
- тест после урока и AI-анализ ошибок по навыкам.

## Запуск

Откройте `index.html` напрямую или запустите локальный сервер из папки `app`:

```bash
python3 -m http.server 4173
```

Затем откройте:

```text
http://localhost:4173/
```

## Материалы

Прототип использует сохраненные учебники как источник тем и ключевых слов. В интерфейсе показаны не файлы, а готовые интерактивные уроки, игровые задания, тесты и анализ результата.
